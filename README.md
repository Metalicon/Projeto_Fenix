# 🔥 Projeto Fênix

> Sua melhor versão começa hoje.

Projeto Fênix é um aplicativo desenvolvido em Flutter para auxiliar pessoas em sua jornada de emagrecimento, organização de treinos, alimentação e criação de hábitos saudáveis.

Este projeto nasceu durante a minha própria transformação física, registrando toda a evolução desde os **130 kg** até a meta final de **90 kg**.

---

## 🎯 Objetivos

- Acompanhar evolução do peso
- Registrar treinos
- Controlar consumo de água
- Organizar alimentação
- Monitorar medidas corporais
- Registrar fotos de evolução
- Criar hábitos saudáveis
- Utilizar gamificação para manter a motivação
